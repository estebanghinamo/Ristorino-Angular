Ristorino es una aplicación web desarrollada en Angular que permite a los usuarios explorar restaurantes, visualizar promociones vigentes y gestionar reservas de mesas de forma intuitiva.

Este repositorio contiene exclusivamente el código del cliente (Frontend). La API REST (Backend) desarrollada en Spring Boot se encuentra en el repositorio [ristorino-rest].

## ✨ Características Principales

* **Exploración de Restaurantes y Promociones:** Interfaz dinámica para visualizar el catálogo de locales gastronómicos.
* **Gestión de Sesiones Seguras:** Sistema de autenticación de usuarios y clientes para mantener el estado de la sesión y proteger rutas privadas.
* **Internacionalización (i18n):** Cambio de idioma dinámico desde la interfaz (implementado con botones de selección de idioma) para mejorar la accesibilidad del usuario.
* **Diseño Responsivo:** Adaptable a diferentes dispositivos (Móvil, Tablet, Desktop).
* **Búsqueda Inteligente (AI-Powered):** Interfaz conectada a un motor de IA (Gemini) que permite a los usuarios realizar búsquedas naturales y complejas, interpretando la intención más allá de simples palabras clave.
